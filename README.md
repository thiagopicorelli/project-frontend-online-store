# FrontEnd Online Store
Trabalho feito em grupo na Trybe de uma aplicação do zero de uma loja online simplificada, acessando a API do Mercado Livre, listando os produtos, e permitindo o usuário a filtrá-los, adicionar eles ao carrinho e também os avaliar localmente. Testado por arquivos de teste do RTL e ESLint.
